# Code of Conduct

本项目采用 Contributor Covenant 的核心原则：友善、尊重、包容、建设性沟通。

## Expected behavior

- 尊重不同经验水平、背景和观点。
- 用可复现信息讨论问题。
- 聚焦代码、文档和用户体验，不做人身攻击。
- 发现安全或隐私问题时优先保护用户数据。

## Unacceptable behavior

- 骚扰、歧视、侮辱或威胁。
- 发布他人的隐私信息、cookies、API key 或敏感任务数据。
- 在 issue / PR 中持续偏离主题或恶意刷屏。

维护者可以编辑、隐藏或关闭违反准则的内容，并在必要时限制参与权限。
